Quaduino

This is the code of a quadcopter, and its RemoteController. I develope those during my homebrew quadcopter build. Both of the on board and hand-held controllers use Arduino Uno Board as controller.

Setting of the quad is + mode and use PID controller for Pitch,Roll angular rate control. And a PD controller for Pitch,Roll angular position control.

This code should work on any hardware, only needed is arduino, modules, and some tweaking about PID gain.

List of modules I am using:

  • nRf24L01+
  • MPU6500

List of Library I am using:

  • RF24 -- https://github.com/maniacbug/RF24
  • i2cdevlib -- https://github.com/jrowberg/i2cdevlib/tree/master/Arduino/MPU6050
